Glosario para un Sistema de Información

Almacenamiento

Guardar la información ya procesada de forma ordenada y segura para poder usarla más adelante.

Componentes

Todas las partes que forman un sistema de información: el hardware, el software, los datos y la gente que lo usa.

Dato

El pedacito más pequeño de información, un valor o un hecho sin contexto completo.

Distribución

Enviar la información procesada a las personas, equipos o sistemas que la necesitan.

Entropía

En los sistemas, se refiere a la tendencia natural a que el desorden aumente con el tiempo.

Homeostasis

La capacidad de un sistema para mantenerse en equilibrio y autorregularse, incluso si hay cambios externos.

Información

Datos que ya han sido procesados, organizados y tienen un significado y propósito, útiles para decidir.

Procesamiento

La acción de transformar datos brutos en información útil con sentido.

Sistema

Un conjunto de partes que se conectan y dependen entre sí para lograr una meta común.

Sistema de Información (SI)

La combinación organizada de personas, hardware, software y datos que ayuda en las operaciones y decisiones de una organización.

Sistema de Información Gerencial (MIS)

Un tipo de SI que apoya a los mandos intermedios para tomar decisiones.

Sistema de Soporte de Decisiones (DSS)

Un tipo de SI que ayuda a la alta dirección a resolver problemas más complicados.

Sistemas Conceptuales

Sistemas hechos de ideas o conceptos abstractos, como las matemáticas o un lenguaje.

Sistemas Materiales

Sistemas hechos de cosas físicas, como nuestro cuerpo o una máquina.

Sistemas Híbridos

Sistemas que mezclan ideas y cosas físicas, como una computadora (hardware + software).

Totalidad

La idea de que en un sistema, todas las partes funcionan como un solo conjunto; si algo cambia en una parte, afecta a todo el sistema.

TPS (Sistemas de Procesamiento de Transacciones)

Un tipo de SI que se encarga de las operaciones y transacciones diarias de una organización.
